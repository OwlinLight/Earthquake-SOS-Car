# Earthquake-SOS-Car

小车通过巡线模块沿线行驶，到达地震现场，当小车通过摄像设备探测识别到人脸时，在原地利用七彩灯和蜂鸣器装置发送SOS救助信号，在结束所有遇灾人员的探测和求助后，小车原路返回，返回到出发位置。重复执行小车求助，直到巡线结束。本次实验设计了1个救助点，中间过程调用1次人脸识别模块。
